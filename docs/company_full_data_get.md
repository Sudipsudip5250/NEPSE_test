# `company_full_data_get.py`

## Purpose
Utility to download or assemble full historical datasets for a single company (used for one-off exports or rebuilding a company's CSV).

## Dependencies
`pandas`, `requests`, `selenium` (if it scrapes), `python-dotenv`

## Run
```bash
python company_full_data_get.py
```

## Notes
- Use this when you need a complete rebuild of a single company's CSV.
